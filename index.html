Changelog
v5.1
Added Best Split and Best Corners buttons. Both use the same sector/stats algo and coin count as Best Coins, placing chips on the top-scoring 2-number splits or 4-number corners within the hot sector or arc. Default Shift applies to all three Best bet types. Zones are selected greedily with no overlapping numbers — each number appears in at most one picked zone.
v5.0
Rebuilt Best Coins algo around physical wheel sectors (Voisins, Tiers, Orphelins) with arc fallback. Added Stats algo (scores numbers by category edge) with Sector/Stats toggle in the sidebar. Added Auto Update Coins toggle for live coin placement on new rounds. Added Default Shift input to offset coin placement by a fixed amount. Moved pattern controls and algo toggle into the sidebar. Input box now always scrolls to show the latest number on the right.
